# Blog Management System

## Features
- User Registration
- User Login
- Dashboard
- CRUD Operations
- Search
- Pagination
- Bootstrap UI
- Password Hashing
- Prepared Statements

## Technologies
- PHP
- MySQL
- Bootstrap 5
- HTML
- CSS
- JavaScript

## How to Run
1. Copy project to htdocs
2. Start Apache and MySQL
3. Import blog.sql
4. Open http://localhost/blog-project
